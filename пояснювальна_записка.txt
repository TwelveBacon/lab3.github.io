У ході виконання лабораторної роботи №3 було створено мобільну версію головної сторінки веб-сайту на основі варіанту №16 з попередніх лабораторних робіт. 
Мета роботи полягала в адаптації сторінки для зручного використання на смартфонах, забезпеченні коректного відображення елементів та створенні сучасного і зручного інтерфейсу для мобільних пристроїв.

1. Реалізація мобільної версії:
  Сторінка була зверстана з використанням HTML і CSS. Усі стилі винесені в окремий файл index_mobile.css для забезпечення зручності управління кодом.
  Використано адаптивний підхід: додано медіа-запити, які дозволяють змінювати розташування та стиль елементів залежно від ширини екрану (до 600px).

2. Основні елементи сторінки:
  Заголовок: виділено великим шрифтом і розташовано по центру з використанням стилю Flexbox.
  Меню: організовано у вигляді вертикального списку, стилізованого для зручної навігації. Кожен пункт меню має підсвітку при наведенні.
  Контент: текстовий блок із основною інформацією збережено без змін, але адаптовано для мобільного перегляду.
  Футер: включає копірайт, стилізований у відповідності до загального дизайну сторінки.

3. Адаптація під мобільні пристрої:
  Застосовано вертикальне розташування елементів для зручного перегляду.
  Встановлено відступи між елементами для уникнення візуального перевантаження.
  Збільшено розмір шрифтів для покращення читабельності на малих екранах.

4. Інтеграція мобільної версії:
  Мобільна версія інтегрована з головною сторінкою, створеною в лабораторній роботі №2. Для цього додано посилання на CSS-файл index_mobile.css в HTML-документі.

5. Результат:
  Мобільна версія успішно протестована на різних екранах смартфонів і забезпечує коректне відображення всіх елементів.
  Головну сторінку разом із мобільною версією опубліковано на GitHub Pages, що дозволяє оцінити її роботу як на ноутбуці, так і на смартфоні.
